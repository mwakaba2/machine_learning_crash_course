# Machine Learning Crash Course
Interactive Machine Learning Applications

## Requirements
* pip
* conda
* python 3.6

## Installation

```bash
conda create --name ml_cc_env python=3.6 # create environment
source activate ml_cc_env
pip install -r requirements.txt
```
